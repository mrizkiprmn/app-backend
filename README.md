### npm Installation

#### Using express
```bash
$ npm install express
```
#### Using npm start
```bash
$ npm install start
```
#### Using eslint
```bash
$ npm install i eslint
```
#### Using body-parser
```bash
$ npm install i body-parser
```
#### Using morgan
```bash
$ npm install i morgan
```
#### Using pg for connect to PostgreSQL
```bash
$ npm install pg
```
#### Using dotenv
```bash
$ npm install dotenv
```

#### Using JWT
```bash
$ npm install jsonwebtoken
```

#### Using bcrypt
```bash
$ npm install bcrypt
```

#### Using jwt-decode
